Project Type: Web Mapping and Calgary Open Data Integration

This demo loads Calgary open-data GeoJSON layers online and displays them with custom styling, popups, and layer controls.

Overview

This project demonstrates web mapping knowledge using Leaflet, an open-source JavaScript mapping library. The map is focused on Calgary and combines a basemap with online GeoJSON feature layers from the City of Calgary open data portal to show how spatial data can be loaded, styled, and explored directly in the browser.

Methods

  • Initialized an interactive Leaflet map centered on Calgary with OpenStreetMap tiles.
  • Loaded Calgary Ward Boundaries and the Calgary LRT centre line as online GeoJSON using fetch().
  • Styled polygon and line layers differently to separate electoral areas and transit infrastructure.
  • Added popups and a layer control to improve map interactivity and usability.

Outcome

The result is a browser-based Calgary map that highlights practical web map programming skills, including layer management, feature interaction, asynchronous data loading, and local open-data integration for a more relevant portfolio example.
