Project Type: Mobile GIS Workflow

Overview

This project focused on building a GIS-based mobile workflow for field data collection using ESRI Field Maps. The goal was to improve consistency, reduce input errors, and make data capture more efficient during fieldwork.

Methods

  • Configured a mobile-ready data collection form in ESRI Field Maps.
  • Added custom attributes and domains to standardize inputs.
  • Designed the workflow to support accurate and repeatable field surveys.
  • Tested the interface from a user perspective in a field collection context.

Outcome

The final workflow improved consistency in recorded attributes and made field data entry more structured. This project highlights practical GIS application design for real-world data collection.
